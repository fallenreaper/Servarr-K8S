# Cloudflared Tunnel

This is a tool used to create a tunnel to cloudflare so you are able to deploy publically. Working through information on cloudflare.com will let you manage specifics related to what and how it is deployed, but this will expose what is needed.

## Setup

To set this up you will need to set up your token which is referenced in the secret. As long as it is set, you can apply this information and it will deploy the secret and the tunnel pod
